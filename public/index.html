Settings & API Docs

General

ReelKeeper stores components, import batches, and movement history in the Docker volume at data/reelkeeper.json.

Part matching

ReelKeeper matches imported rows by LCSC part number first, then manufacturer part number, then compatible passive specs where safe.

LCSC pricing

Refresh USD price breaks for every component with an LCSC part number. BOM estimates use the price break for the quantity needed on one board.

Pricing has not been updated in this session.

Reset ReelKeeper

Remove all components, stock movements, and order import history and return the software to its original empty state.

Use component

Tell ReelKeeper which component was used and how many. Identify it with an LCSC part number, MPN, or internal component id.

curl -X POST http://localhost:3000/api/use \
  -H "Content-Type: application/json" \
  -d '{"lcsc":"C25804","quantity":1}'

Order import API

POST /api/import/order
{
  "rows": [
    {
      "lcsc": "C25804",
      "mpn": "0603WAF1002T5E",
      "name": "10k resistor",
      "quantity": 1000,
      "category": "Resistors",
      "package": "0603"
    }
  ]
}

BOM check API

POST /api/bom/check
{
  "rows": [
    { "designator": "R1 R2", "lcsc": "C25804", "quantity": 2 }
  ]
}
